# SOC-Advanced-Toolkit
Python öğrenme sürecimin bir parçası olarak geliştirdiğim; modüler mimariye ve asenkron tarama özelliklerine sahip temel bir siber güvenlik toolkit


# Repoyu klonlayın
git clone https://github.com/beytullaharslannn/SOC-Advanced-Toolkit

# Proje dizinine girin
cd SOC-Advanced-Toolkit

# Gerekli bağımlılıkları yükleyin
pip install -r requirements.txt

# Çalıştırın
python3 main.py
